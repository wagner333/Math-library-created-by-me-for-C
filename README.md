# Projeto: Operações Matemáticas em Assembly com C

## Descrição

A ideia do projeto é bem direta: implementar **operações matemáticas básicas em Assembly** (como soma, subtração, multiplicação, divisão, potência, fatorial, entre outras) e **chamá-las a partir de um programa em C**.

O código em C fica responsável pela lógica principal da aplicação, bem como pela **entrada e saída de dados**, enquanto o Assembly é utilizado exclusivamente para realizar os **cálculos matemáticos**.

## Objetivo

O principal objetivo deste projeto é demonstrar, de forma prática e simples:

- A integração entre **C e Assembly**
- A passagem de parâmetros entre as linguagens
- O retorno de valores de funções em Assembly
- O funcionamento de operações em **baixo nível**, próximas ao hardware

## Abordagem

- As funções matemáticas são implementadas em Assembly
- Essas funções são declaradas e utilizadas no código em C
- O programa em C faz as chamadas, exibe os resultados e controla o fluxo da aplicação

Este projeto serve como um exercício educacional para quem deseja entender melhor como linguagens de alto e baixo nível podem trabalhar juntas.
